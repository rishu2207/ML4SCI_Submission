# ML4SCI Hackathon Submission

Welcome to my submission for the ML4SCI Hackathon! This repository contains the complete workflow and solutions for two major tasks, packaged into a fully reproducible and documented setup.

## Tasks & Deliverables

### [Task 1: Agentic AI Simulation](Task1_Agentic_AI/)
This task covers a modular Python-based simulation using Agentic AI workflows. The entire logic has been consolidated into a single Jupyter Notebook (`Agentic_AI.ipynb`) for easy review by the judges, along with its dedicated README.
- **Location:** `Task1_Agentic_AI/`
- **Main Notebook:** `Agentic_AI.ipynb`

### [Task 2: Gravitational Lensing Image Classification](Task2_Classification/)
This task involves multi-class classification (No Substructure vs Subhalo vs Vortex) of strong gravitational lensing images using a custom PyTorch Convolutional Neural Network.
- **Location:** `Task2_Classification/`
- **Main Notebook:** `Classification.ipynb`
- **Features:** Comprehensive Exploratory Data Analysis, Dual-Phase Training loop, Confusion Matrix, and ROC Curve evaluation metrics.
